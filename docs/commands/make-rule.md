# `beyond:make:rule`
Creates a new rule. 

## Signature
`beyond:make:rule {name} {--g|global} {--force}`

| Parameters | Description           |
|------------|-----------------------|
| name       | The name of your rule |

| Flags    | Description                                  |
|----------|----------------------------------------------|
| --global | Creates a global rule inside `Support\Rules` |
| --force  | Overwrite existing file                      |
