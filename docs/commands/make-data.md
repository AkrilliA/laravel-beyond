# `beyond:make:data`
Creates a new Spatie data transfer object.

> [!IMPORTANT] 
> Requires package `spatie/laravel-data`

## Signature
`beyond:make:data {name} {--force}`

| Parameters | Description                           |
|------------|---------------------------------------|
| name       | The name of your data transfer object |

| Flags   | Description             |
|---------|-------------------------|
| --force | Overwrite existing file |
