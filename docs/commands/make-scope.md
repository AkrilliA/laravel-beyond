# `beyond:make:scope`
Creates a new scope. 

## Signature
`beyond:make:scope {name} {--g|global} {--force}`

| Parameters | Description            |
|------------|------------------------|
| name       | The name of your scope |

| Flags    | Description                               |
|----------|-------------------------------------------|
| --global | Creates the scope inside `Support\Scopes` |
| --force  | Overwrite existing file                   |
