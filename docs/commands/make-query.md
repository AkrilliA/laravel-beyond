# `beyond:make:query`
Creates a new Spatie query builder.

> [!IMPORTANT] 
> Requires package `spatie/laravel-query-builder`

## Signature
`beyond:make:query {name} {--force}`

| Parameters | Description                    |
|------------|--------------------------------|
| name       | The name of your query builder |

| Flags   | Description             |
|---------|-------------------------|
| --force | Overwrite existing file |
